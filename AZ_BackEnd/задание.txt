ЛР№2: Разработка серверной части

Задание: Разработать Web API в ASP.NET Core

+ 1. Описать модель:
+ 2-3 класса;
+ классы должны содержать данные и методы для 
  работы с ними;

2. Добавить в API контроллеры для работы с моделью
+ CRUD-операции;
- 2-5 запросов на объединение данных из
  разных классов;
- использовать LINQ для работы с коллекциями
  (Where и Select - программа минимум, можно добавить и другие);

+ 3. Добавить авторизацию:
+ получение сводных данных только для авторизованных
  пользователей;
+ добавление данных - только для администраторов;

4. Создать репозиторий:
- выложить туда код;
- прислать ссылку;

5. Собрать релиз-версию проекта:
- скинуть архив с релизом;
- скинуть коллекцию запросов для тестирования.
